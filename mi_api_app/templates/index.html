News Public API

API de uso público para obtener todas las noticias de último momento. Esta API fue creada desde 0 utilizando Python REST FRAMEWORK, si este proyecto te fue de utilidad no olvides de dar ⭐ en github!

Esquema JSON

Schema

{
  "title": "Título de la noticia",
  "description": "Descripción de la noticia",
  "site_icon": "URL del icono del sitio",
  "link": "URL de la noticia",
  "image": "URL de la imagen"
}

Endpoint

GET https://news-public-api.onrender.com/api/noticias/?all=true

Limit

GET https://news-public-api.onrender.com/api/noticias/?all=true

?all=true : Entrega todas las noticias

GET https://news-public-api.onrender.com/api/noticias/?limit=YOUR LIMIT

?limit=YOUR LIMIT : Entrega la cantidad de noticias indicada

Ejemplo Python

Python

import requests

url = "https://news-public-api.onrender.com/api/noticias/?all=true"
response = requests.get(url)
data = response.json()

Ejemplo JavaScript

JavaScript

const url = "https://news-public-api.onrender.com/api/noticias/?all=true";

fetch(url)
  .then(response => response.json())
  .then(data => console.log(data));
